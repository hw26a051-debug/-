# ブロック仕様書 (blocks.md)

## 1. ブロックの基本構成
- **構成要素**: テトリミノのように、正方形のセルが連結した形状を持つ。
- **文字の配置**: 正方形のセル1つにつき、単語の「ひらがな1文字」が配置される。
  - 例：3文字の単語「さくら」の場合、3つの正方形セルが連結し、それぞれのセルに「さ」「く」「ら」が順番に格納される。

## 2. 形状と生成ルール
- **基本形状**:
  - **直線型（棒状）**: 基本となる形状。
  - **L字型**: 唯一許容される屈折形状。ランダムで生成される。
- **単語の長さ（文字数）**:
  - 最小文字数：**3文字**
  - 最大文字数：**7文字**
- **配置方向**:
  - 縦向き、または横向きのいずれかで生成される。
  - **5文字以上の単語**（5〜7文字）については、視認性を確保するため**必ず横向き**で生成・落下させる。
- **出現位置（X座標）**:
  - 画面上部に出現する際、偏りをなくすために**左右のX座標はランダムに決定**される（枠外にはみ出さないよう調整）。

## 3. デザインと視認性
- **セル境界線**: ブロック同士の境目やセル個々の輪郭がはっきりと見やすくなるよう、高コントラストな枠線と個別のカラーリング（カラフルなネオンカラー等）を適用する。

## 4. レベルデザイン（落下速度）
- ブロック自体の落下速度は、ゲームの経過時間または消去ブロック数に応じて、段階的に高速化していくものとする。
